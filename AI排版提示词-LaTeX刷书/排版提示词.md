### **详细排版规则**
#### **1. 章节与标题**
*   **带编号章节:** 用于主要的章节标题，如 `§1 实数`。
    ```latex
    \section{§1 实数}
```
*   **不编号子节:** 用于节内的标题，如 `一、实数及其性质`。
    ```latex
    e   \section*{一、实数及其性质}
    ```
*   **特殊标题 (习题/总练习题):** 这是最重要的自定义样式，必须严格遵守。用于每节末尾的习题部分或章末的总结练习。
    *   **效果要求:** 居中、大号字体、加粗，并自动添加到PDF目录中且可点击跳转。
    *   **标准代码模板:**
        ```latex
        \vspace{1cm}
        \phantomsection
        \addcontentsline{toc}{section}{习题 1.1} %<-- 此处的标题文字需根据实际情况修改
        {\centering\LARGE\bfseries 习题 1.1\par} %<-- 此处的标题文字需根据实际情况修改
        \vspace{0.3cm}
        ```
    *   **说明:**
        *   `\vspace{1cm}`: 与正文内容保持足够距离。
        *   `\phantomsection`: 创建跳转锚点，保证目录链接准确。
        *   `\addcontentsline{toc}{section}{...}`: 将标题手动添加到目录。
        *   `{\centering\LARGE\bfseries ...\par}`: 实现居中、大号、加粗的视觉效果。
#### **2. 内容环境 (Environments)**
所有定义、定理、证明、例子等内容都必须用对应的环境包裹。
*   **定义 (Definition):**
    ```latex
    \begin{definition}[可选的定义标题或编号]
        ...
    \end{definition}
    ```
*   **定理 (Theorem):**
    
    ```latex
    \begin{theorem}[可选的定理标题或编号]
        ...
    \end{theorem}
    ```
*   **命题 (Proposition):**
    
    ```latex
    \begin{proposition}[可选的命题标题或编号]
        ...
    \end{proposition}
    ```
*   **引理 (Lemma):**
    ```latex
    \begin{lemma}[可选的引理标题或编号]
        ...
    \end{lemma}
    ```
*   **推论 (Corollary):**
    
    ```latex
    \begin{corollary}[可选的推论标题或编号]
        ...
    \end{corollary}
    ```
*   **示例 (Example):**
    
    ```latex
    \begin{example}[可选的示例标题或编号]
        ...
    \end{example}
    ```
*   **证明 (Proof):**
    
    ```latex
    \begin{proof}
        ...
    \end{proof}
    ```
*   **备注 (Remark):**
    ```latex
    \begin{remark}
        ...
    \end{remark}
    ```
*   **练习 (Exercise):** 这是习题部分的核心。
    *   每一道独立的习题（即使只有一个小问题）都必须用 `\begin{exercise}...\end{exercise}` 包裹。
    *   **关键规则:** 在每个 `exercise` 环境之间，必须使用 `\vspace{1cm}` 来创造清晰的垂直间距。
    *   **示例:**
        ```latex
        \begin{exercise}
            1. 这是第一道题...
        \end{exercise}
        
        \vspace{1cm}
        
        \begin{exercise}
            2. 这是第二道题...
        \end{exercise}
        ```
#### **3. 图片处理**
*   图片和其下方的图注标题应分别居中。
*   **标准代码模板:**
    ```latex
    \begin{center}
        \includegraphics[width=0.4\textwidth]{path/to/your/image.jpg}
    \end{center}
    \begin{center}
        图 1-1 %<-- 图注标题
    \end{center}
    ```
#### **4. 段落与缩进**

*   常规的中文段落自动首行缩进。
*   对于明确不需要缩进的行，如以“**注**”、“**解**”或某些列表项开头的段落，应在该行行首添加 `\noindent` 命令。
    ```latex
    \noindent 注 这是一个不需要首行缩进的注释段落。
    ```
#### **5. 脚注**

*   **必须**使用 LaTeX 的标准脚注命令 `\footnote{...}`。
*   严禁使用手动上标符号（如 `①`）或非标准的自定义命令来创建脚注。
#### **6. 数学公式**
*   **行内公式:** 使用 `\( ... \)`。
*   **行间公式 (不编号):** 使用 `\[ ... \]`。
*   **行间公式 (手动编号):** 在 `\[ ... \]` 环境内部使用 `\tag{...}`。
    ```latex
    \[
    a^2 + b^2 = c^2 \tag{3}
    \]
    ```
